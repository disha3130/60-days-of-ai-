To understand the difference between a basic prompt and an engineered prompt while working with AI image generation.

## Engineered Prompt Used

Requirements:

* Title: "Prompt Engineering"

* Include 4 sections:

  1. What is a Prompt?
  2. Why Prompt Engineering Matters

  User → Prompt → AI Model → Response

## Comparison

### Lazy Prompt Output

* Generic explanation
* Limited structure
* Fewer details
* Basic visual design

### Engineered Prompt Output

* Clear structure
* Better visual hierarchy
* More detailed explanations
* Included examples and flow diagrams
* Professional infographic design

## Key Learnings

* Detailed prompts produce higher-quality outputs.
* Clear instructions improve accuracy.
* Providing structure helps AI organize information effectively.
* Prompt Engineering is important for obtaining useful AI responses.
