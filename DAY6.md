# Day 6 – Resume Optimizer

## Objective

The goal of this task was to learn how AI can analyze and optimize a resume for Applicant Tracking Systems (ATS) and improve its chances of getting shortlisted.

## Process Followed

1. Uploaded my resume to Claude.
2. Used the Resume Optimizer prompt.
3. Reviewed the ATS analysis report.
4. Identified missing keywords and formatting issues.
5. Generated an optimized one-page ATS-friendly resume.
6. Saved the optimized version and documented the results.

## ATS Analysis Summary

* Evaluated resume structure and formatting.
* Checked ATS compatibility.
* Identified missing keywords relevant to target roles.
* Suggested improvements for skills, projects, and achievements.

## Improvements Made

* Improved resume formatting for ATS readability.
* Added relevant technical keywords.
* Enhanced project descriptions using action verbs.
* Highlighted important skills and achievements.
* Created a concise one-page resume.

## Key Learnings

* ATS systems scan resumes for keywords and proper formatting.
* Strong project descriptions improve resume quality.
* Quantifiable achievements make resumes more impactful.
* A well-structured one-page resume is preferred for freshers.
* AI tools can significantly improve resume quality and ATS compatibility.

## Conclusion

This exercise helped me understand how ATS systems evaluate resumes and how AI can be used to create a stronger, more professional resume for internship and job applications.
